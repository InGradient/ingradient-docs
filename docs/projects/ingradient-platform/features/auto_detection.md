# 자동 감지 / 추론 (Auto-Detection & Inference)

## 기능 개요

프로젝트에 배정된 AI 모델을 사용하여 이미지에서 객체를 자동 감지하거나 분류하는 기능이다. 라벨링 작업을 보조하는 핵심 AI 기능이다.

## 문제 정의

수만 장의 이미지를 수동으로 라벨링하는 것은 비효율적이다. 사전 학습된 모델이나 프로젝트에서 학습한 모델로 자동 감지한 결과를 사용자가 확인/수정하면 라벨링 속도를 크게 높일 수 있다.

## 범위 / 비범위

- **포함**: 이미지별 객체 감지 실행, 결과 오버레이, 신뢰도 필터링, 모델 선택
- **제외**: 모델 학습 → Training, 모델 다운로드 → Models Library

## 사용자 시나리오

### 자동 감지 실행

- **기본 흐름**:
  1. Gallery 상세 모달에서 "자동 감지" 클릭
  2. 프로젝트에 배정된 모델(object_detection 또는 classification)로 추론
  3. 결과를 바운딩 박스 + 신뢰도 점수로 오버레이
  4. 사용자가 확인 → 수정/삭제/추가 → 저장
- **예외 흐름**:
  - 모델 미배정 → "모델을 먼저 배정하세요" 안내
  - 추론 실패 → 에러 표시

### 결과 활용

- 바운딩 박스: 정규화 좌표 (0~1), 클래스 할당, 신뢰도 점수
- 히트맵: 모델 출력의 활성화 영역 시각화
- `ranInference` 플래그로 모델 추론 여부 추적

## 데이터 / API / 권한 영향

| API | 용도 |
|-----|------|
| `POST /api/projects/{id}/detect` | 객체 감지 실행 |

- 응답: `{ bboxes: [{x, y, w, h, classId, confidence}], ranInference: boolean }`
- Models Library에서 `object_detection` 또는 `classification` task에 모델을 배정해야 사용 가능

## 테스트 계획

- 모델 배정 → 이미지에서 자동 감지 실행 → 바운딩 박스 표시
- 신뢰도 점수 표시 확인
- 모델 미배정 시 안내 메시지
- 감지 결과 수정 후 저장 → 라벨로 반영
