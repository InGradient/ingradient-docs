### 좌측 — 이미지 캔버스

- 원본 이미지 + 바운딩 박스/포인트 오버레이
- 드래그로 bbox 그리기, 클릭으로 포인트 찍기
- 줌/패닝 지원
- AI 감지 결과 오버레이 (신뢰도 점수 표시)

### 우측 사이드바 (리사이징 가능)

| 섹션 | 내용 |
|------|------|
| Classes | 클래스 목록 + 체크박스 (표시/숨김) + 어노테이션 수 |
| Info | 해상도, 크기, 날짜, 업로더, 소스, Split, Sync 상태 |
| Comments | 코멘트 스레드, @멘션, 입력 필드 |

### 도구 모음

| 도구 | 기능 |
|------|------|
| Cursor | 선택/이동 |
| Bbox | 바운딩 박스 그리기 |
| Point | 포인트 찍기 |
| Class | 이미지 분류 라벨 |
| Undo/Redo | 실행 취소/다시 |
| Clear | 어노테이션 전체 삭제 |
| Zoom | 확대/축소 |

### 탐색

- `←` `→` 화살표로 이전/다음 이미지
- 시퀀스 그룹 내 연속 탐색
