### 좌측 패널 — 데이터셋

- 데이터셋 목록 (체크박스 선택)
- 요약: 총 이미지 수, Train/Eval/Test 카운트
- 컨텍스트 메뉴 (⋯): 복사, 이름 변경, 삭제, 내보내기

### 중앙 — 갤러리 그리드

**툴바:**
```
┌──────────────────────────────────────────────────┐
│ ☐ 3 of 1,240 │ Clear │ 🗑 Del │ Grid▾ │ 🔍 │ ▼Filter │ ▼Sort │
└──────────────────────────────────────────────────┘
```

- 드래그앤드롭 이미지 업로드 (영역 전체)
- 가상화 그리드 (수십만 장 대응)
- 이미지 선택: 클릭, Ctrl+클릭, Shift+범위
- 뷰 모드: Grid / Table / Stats

### 우측 패널 — 클래스 & 멤버

- 프로젝트 클래스 풀 (색상 스와치 + 이미지 수)
- 클래스 검색
- 프로젝트 멤버 목록 (역할 표시)

### 모달

| 모달 | 트리거 | 내용 |
|------|--------|------|
| 데이터셋 추가 | [+ Add] | 이름 입력 → 생성 |
| 데이터셋 복제 | 컨텍스트 메뉴 | 새 이름 + task 타입 |
| 이미지 이동/복사 | 드래그 또는 일괄 작업 | 대상 데이터셋 선택, 이동/복사 옵션 |
| 삭제 확인 | 🗑 버튼 | 대상 수 표시, 확인/취소 |
