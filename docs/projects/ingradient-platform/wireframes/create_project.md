# 프로젝트 생성

## 전체 레이아웃 (`/create-project`)

로그인과 동일한 중앙 정렬 카드 레이아웃.

```
┌──────────────────────────────────────────────┐
│                                              │
│              [ INGRADIENT Logo ]              │
│                                              │
│  ┌────────────────────────────────────────┐  │
│  │          Add Project                   │  │
│  │                                        │  │
│  │  Project name                          │  │
│  │  [                                  ]  │  │
│  │                                        │  │
│  │  Description                           │  │
│  │  ┌──────────────────────────────────┐  │  │
│  │  │                                  │  │  │
│  │  └──────────────────────────────────┘  │  │
│  │                                        │  │
│  │  First dataset name (optional)         │  │
│  │  [                                  ]  │  │
│  │                                        │  │
│  │  Project type                          │  │
│  │  ○ General                             │  │
│  │  ○ Deflectometry                       │  │
│  │                                        │  │
│  │  ┌──────────────────────────────────┐  │  │
│  │  │                                  │  │  │
│  │  │  📁 Drag images here             │  │  │
│  │  │     or click to select           │  │  │
│  │  │                                  │  │  │
│  │  └──────────────────────────────────┘  │  │
│  │                                        │  │
│  │  ┌──────────────────────────────────┐  │  │
│  │  │         Create Project           │  │  │
│  │  └──────────────────────────────────┘  │  │
│  │  [Cancel]                              │  │
│  └────────────────────────────────────────┘  │
│                                              │
└──────────────────────────────────────────────┘
```

### 상태

| 상태 | 표시 |
|------|------|
| 기본 | 폼 활성화 |
| 로딩 | 버튼 "Creating…", 폼 비활성화 |
| 에러 | 상단 에러 Alert |
| 이미지 선택됨 | 드롭존 아래 파일 목록 (썸네일 + 파일명 + 크기 + 삭제 버튼) |
