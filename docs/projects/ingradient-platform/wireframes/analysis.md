# 분석 (Analysis)

## 전체 레이아웃 (`/analysis`)

Dashboard와 동일한 위젯 세트에 프로젝트 선택기가 추가된 형태.

```
┌──────────────────────────────────────────────────────┐
│ Data Analysis                     [Project ▾ demo  ] │
│ Review dataset metrics and annotation quality        │
├──────────────────────────────────────────────────────┤
│                                                      │
│  (Dashboard와 동일한 8종 위젯 그리드)                  │
│                                                      │
│  ┌──────────────────┐  ┌──────────────────────────┐  │
│  │ Data Collection  │  │ Timeline                 │  │
│  └──────────────────┘  └──────────────────────────┘  │
│  ┌──────────────────┐  ┌──────────────────────────┐  │
│  │ Labeling Status  │  │ Class Distribution       │  │
│  └──────────────────┘  └──────────────────────────┘  │
│  ...                                                 │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### Dashboard와의 차이

| 항목 | Dashboard (`/`) | Analysis (`/analysis`) |
|------|-----------------|----------------------|
| 프로젝트 | 현재 선택 프로젝트 고정 | 우측 상단 드롭다운으로 전환 가능 |
| GPU/큐 | 상단에 GPU 상태 + 학습 큐 표시 | 없음 |
| 위젯 | 동일 8종 | 동일 8종 |
| 레이아웃 | 사용자별 커스터마이징 | 기본 레이아웃 |

### 상태

| 상태 | 표시 |
|------|------|
| 프로젝트 미선택 | "프로젝트를 선택하세요" 안내 |
| 로딩 중 | "Loading analysis…" |
| 에러 | Alert 에러 메시지 |
| 데이터 없음 | 각 위젯에 빈 상태 표시 |
