# 로그인 / 회원가입

## 로그인 (`/login`)

전체 화면 중앙 정렬. 최대 너비 420px.

```
┌──────────────────────────────────────────────┐
│                                              │
│              [ INGRADIENT Logo ]              │
│                                              │
│  ┌────────────────────────────────────────┐  │
│  │           Sign in                      │  │
│  │                                        │  │
│  │  ┌──────────────────────────────────┐  │  │
│  │  │ Email                            │  │  │
│  │  └──────────────────────────────────┘  │  │
│  │  ┌──────────────────────────────────┐  │  │
│  │  │ Password                         │  │  │
│  │  └──────────────────────────────────┘  │  │
│  │                                        │  │
│  │  ☐ Keep me signed in   ☐ Remember pw  │  │
│  │                                        │  │
│  │  ┌──────────────────────────────────┐  │  │
│  │  │          Sign in                 │  │  │
│  │  └──────────────────────────────────┘  │  │
│  │                                        │  │
│  │  Don't have an account? Sign up       │  │
│  └────────────────────────────────────────┘  │
│                                              │
└──────────────────────────────────────────────┘
```

### 상태

| 상태 | 표시 |
|------|------|
| 기본 | 폼 활성화 |
| 로딩 | 버튼 "Signing in…", 폼 비활성화 |
| auth 미연결 | 경고 Alert + 폼 비활성화 |
| 인증 실패 | 에러 Alert (폼 상단) |

## 회원가입 (`/signup`)

로그인과 동일한 레이아웃. 필드 추가.

```
┌────────────────────────────────────────┐
│           Sign up                      │
│                                        │
│  [ Email                             ] │
│  [ Name                              ] │
│  [ Organization                      ] │
│  [ Password                          ] │
│                                        │
│  [          Sign up                  ] │
│                                        │
│  Already have an account? Sign in     │
└────────────────────────────────────────┘
```
