### 좌측 사이드바 (220px)

- 클래스 목록 (색상 점 + 이름 + 이미지 수)
- 선택된 클래스 하이라이트
- [+ Add] 클래스 추가 버튼
- [↻] 새로고침 버튼

### 중앙 — 샘플 이미지

- 선택된 클래스의 샘플 이미지 그리드
- 바운딩 박스 오버레이 표시
- 이미지 클릭 → 확대 (라이트박스)
- 우클릭 컨텍스트 메뉴: "참조 이미지로 설정"
- 참조 이미지 드롭존 (하단)

### 우측 패널 (340px, 접기 가능)

**Info 탭:**

| 항목 | 요소 |
|------|------|
| Name | 텍스트 입력 (자동 저장) |
| Color | 색상 스와치 + hex 입력 + 랜덤 버튼 |
| COCO Mapping | YOLO 모델 클래스 매핑 드롭다운 |
| Statistics | 총 이미지, 데이터셋별 분포 |
| Danger | 클래스 삭제 버튼 (빨간색) |

**Description 탭:**

- 텍스트 에어리어 (설명 편집)

### 모달

| 모달 | 트리거 | 내용 |
|------|--------|------|
| 클래스 추가 | [+ Add] | 이름 입력 → 생성 |
| 삭제 확인 | [🗑 Delete] | 경고 + 확인/취소 |
| 라이트박스 | 이미지 클릭 | 전체 화면 이미지 뷰 |
