# ingradient-platform Product Guide

## 제품 개요

ingradient-platform은 산업용 이미지 데이터를 관리하고, AI 모델을 학습시키기 위한 웹 플랫폼이다. 프로젝트 단위로 데이터, 클래스, 모델, 팀원을 관리한다.

## 역할

| 역할 | 설명 |
|------|------|
| Owner | 프로젝트 소유자. 모든 기능 접근 가능 |
| Editor | 데이터 업로드, 라벨링, 학습 요청 가능 |
| Viewer | 데이터/결과 조회만 가능 |

## 화면별 기능 안내

### 로그인 (`/login`)

- Login ID와 비밀번호로 로그인
- 회원가입 (`/signup`) 가능
- auth-service를 통해 인증 후 프로젝트 멤버십 확인

### 대시보드 (`/`)

- 프로젝트 요약 정보 (데이터 수, 학습 상태 등)
- 커스터마이징 가능한 분석 위젯
- 위젯 배치를 드래그앤드롭으로 변경 가능
- 대상: 모든 역할

### 카탈로그 (`/catalog`)

- 데이터셋 목록 확인 및 생성
- 이미지 업로드 (드래그앤드롭, 다중 파일)
- 이미지 검색, 필터링, 정렬
- 데이터셋 간 이미지 복사/이동
- 이미지 일괄 선택 및 삭제
- 대상: Owner, Editor (업로드/삭제), Viewer (조회)

### 갤러리 (`/gallery`)

- 이미지 그리드 뷰
- 이미지 클릭 시 상세 모달 (원본 크기, 메타데이터, 라벨)
- 라벨 확인 및 수정
- 코멘트 작성 (멘션 기능)
- 대상: 모든 역할 (편집은 Editor 이상)

### 클래스 관리 (`/classes`)

- 프로젝트별 클래스 분류 체계 정의
- 클래스 추가/수정/삭제
- 다국어 Display Name (한국어, 영어, 베트남어)
- 클래스별 색상 지정
- 대상: Owner, Editor

### 학습 (`/training`)

- 학습 요청 생성 (데이터셋, 모델 타입, 파라미터 선택)
- 학습 큐 확인 (대기/진행/완료/실패)
- 진행률 모니터링
- 학습 결과 메트릭 확인
- 대상: Owner, Editor

### 모델 (`/models`)

- 학습 완료된 모델 목록
- 모델 버전 관리
- 프로젝트에 모델 배정
- 대상: Owner, Editor (배정), Viewer (조회)

### 분석 (`/analysis`)

- 모델 추론 결과 시각화
- 이미지별 예측 결과 비교
- 대상: 모든 역할

### Edge (`/edge`)

- Edge 디바이스 목록 확인
- 모델 패키지 생성 및 배포 (.ige 파일)
- 디바이스 세션 분석
- 대상: Owner, Editor

### 설정 (`/settings`)

- 프로젝트 정보 수정
- 멤버 초대/제거, 역할 변경
- 역할별 권한 매트릭스 설정
- 대상: Owner (전체), Editor (조회)

### 프로젝트 생성 (`/create-project`)

- 새 프로젝트 생성
- 대상: 인증된 사용자

## 데이터 내보내기

- COCO JSON 형식으로 데이터셋 내보내기
- 선택한 데이터셋/이미지 범위에 따라 ZIP 파일 생성
- 외부 학습 도구 (YOLO, Detectron2 등)와 호환
