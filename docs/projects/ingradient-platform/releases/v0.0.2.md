## 배포 계획

- **migration**: SQLite schema migration 필수 (`_schema_version` 확인)
- **환경변수 (frontend)**: `VITE_IAM_BASE`, `VITE_AUTH_PROJECT_CODE` 필수 설정
- **환경변수 (backend)**: `AUTH_SERVICE_BASE`, `AUTH_SERVICE_PROJECT_CODE` 필수 설정
- **downstream 영향**: auth-service가 먼저 배포되어 있어야 함 (JWT 검증, 멤버십 API)
- **rollback 계획**: `VITE_IAM_BASE` 제거 시 로컬 auth 폴백으로 동작

## 의존성 / 제약사항

- **auth-service v0.0.2+**: JWT 발급, 멤버십 API, Role 체계 필수
- **`@ingradient/ui`**: 디자인 토큰, 컴포넌트 라이브러리 (로컬 링크)
- **ingradient-edge**: Edge 패키지 포맷 (.ige) 호환성 유지 필요
- **ingradient-ai**: GPU 학습/추론 실행 (선택적 — 없으면 학습 기능 미동작)
