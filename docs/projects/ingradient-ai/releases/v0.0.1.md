## 버전 목표

AI control plane의 핵심 기반을 구현한다. utility job end-to-end 동작, 운영 콘솔 5페이지, model registry 기본 CRUD를 달성한다.

## 완료 기준

- utility job 생성 → 실행 → 결과 저장 end-to-end
- failed job triage가 ops console에서 가능
- worker/GPU health 확인 가능
- model version validate/promote-default/rollback 가능
- job detail에 attempt history, routing reason 표시
