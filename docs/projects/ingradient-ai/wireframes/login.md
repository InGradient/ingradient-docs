# 로그인

## 진입 경로

- Ops Console URL 접속 시 (미인증 상태)
- 세션 만료 시 자동 리다이렉트

## 화면 구조

### 전체 레이아웃

- App Shell(Left Nav, Top Bar) 없이 단독 페이지로 표시된다
- 화면 중앙에 로그인 카드가 배치된다

### 로그인 카드

- 상단: INGRADIENT 로고
- Login ID 입력 필드 (placeholder: "Login ID")
- 비밀번호 입력 필드 (placeholder: "Password")
- "Login" 버튼 (기본 활성화, 입력값 없으면 비활성화)

## 주요 인터랙션

- **"Login" 클릭**: auth-service에 인증 요청
  - 성공 + admin 역할 → Overview 페이지로 이동
  - 성공 + admin 아님 → 에러 메시지 "접근 권한이 없습니다. Admin 계정으로 로그인하세요."
  - 실패 (잘못된 ID/PW) → 에러 메시지 "Login ID 또는 비밀번호가 올바르지 않습니다."
  - 계정 잠금 → 에러 메시지 "계정이 잠겼습니다. 관리자에게 문의하세요."
- **Enter 키**: "Login" 클릭과 동일

## 상태별 화면

- 기본: 로그인 카드만 표시
- 로그인 중: "Login" 버튼 로딩 상태 (텍스트: "Logging in...")
- 에러: 로그인 카드 하단에 에러 메시지 (빨간색)
- 세션 만료: 상단에 "세션이 만료되었습니다. 다시 로그인하세요." 안내
