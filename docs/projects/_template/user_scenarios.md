# User Scenarios

- 문서명: User Scenarios
- 목적: 사용자가 특정 목적을 달성하기 위해 어떤 화면을 거치는지 흐름 단위로 정리한다.
- 상태: Draft
- Owner: Product
- 마지막 수정일:

## 이 문서의 역할

- `product_guide.md`가 "각 화면에서 뭘 할 수 있는가"라면, 이 문서는 **"이 목적을 달성하려면 어떤 화면을 거치는가"**를 정리한다
- features/의 상세 시나리오(예외 흐름, 에러 처리, 테스트 계획)까지는 다루지 않는다
- 기술 용어 없이, 사용자 관점의 흐름으로 작성한다

## 작성 형식

목적 단위로 아래를 정리한다.

```markdown
### {목적}

1. 사용자가 ~ 한다
2. ~ 페이지에서 ~ 한다
3. ~ 결과가 된다
```
