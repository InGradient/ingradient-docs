# Manual Tests

- 문서명: Manual Tests
- 목적: 자동화할 수 없어 사람이 직접 확인해야 하는 테스트 항목을 누적 관리한다.
- 상태: Draft
- Owner: QA / Dev
- 마지막 수정일:

## 이 문서의 역할

- 자동화 테스트(unit, integration, e2e)로 커버할 수 없는 항목만 여기에 기록한다
- 기능이 추가될 때마다 수동 테스트 항목을 누적한다. 제거하지 않고 쌓는다
- 릴리즈 전, 또는 관련 기능을 수정했을 때 이 문서를 열고 해당 항목을 직접 확인한다
- 자동화가 가능해진 항목은 자동화 전환 후 이 문서에서 제거한다

## 수동 테스트가 필요한 경우

| 유형 | 예시 |
|------|------|
| 시각적 확인 | 레이아웃, 색상, 폰트, 정렬, 반응형 |
| UX 흐름 | 시나리오대로 처음부터 끝까지 사용해보기 |
| 상태 전환 | empty → loading → success → error 전환이 자연스러운지 |
| 크로스 브라우저/디바이스 | Chrome, Firefox, Safari, mobile |
| 환경 의존 | 오프라인/온라인 전환, 네트워크 끊김, 느린 네트워크 |
| 권한별 화면 | 역할별로 보이는 것/숨겨지는 것이 맞는지 |
| 인쇄/내보내기 | PDF, CSV 등 출력물 확인 |
| 접근성 | 키보드 탐색, 스크린 리더, 포커스 이동 |
