# Releases

## 이 폴더는 무엇인가

버전별 릴리즈 계획을 관리하는 폴더다. 각 버전 파일에는 해당 릴리즈의 목표, 포함할 기능/개선/버그 수정 목록, 진행 상황 체크리스트를 담는다.

## 역할 구분

| 문서 | 시점 | 내용 |
|------|------|------|
| `releases/v{버전}.md` | 배포 전 (기획/개발 중) | 이번 버전에서 뭘 할 건지, 진행 상황 |
| `release_notes.md` | 배포 후 | 실제로 뭘 넣었는지, 운영 영향 |
| `roadmap.md` | 상시 | 전체 방향과 우선순위 |
| `features/{기능명}.md` | 기획 시 | 개별 기능 상세 기획 |
