# 라이선스 & 디바이스

## 진입 경로

- 사이드바 "Licenses & Devices" 클릭 → `/admin/licenses`

## 화면 구조

- 상단:
  - 요약 카드: 총 수, 활성, 온라인, 오프라인
  - 검색바 (조직, 디바이스 이름, UID, 상태)
  - 모드 필터 탭 (All / Online / Offline)
  - "+ 오프라인 라이선스 발급" 버튼
- 메인:
  - 테이블: 조직, 이름/라벨, Device UID/Fingerprint, 모드 (Online/Offline), 상태, 등록일
  - 행마다 액션: 폐기, 삭제 (온라인) / 키 복사, 폐기 (오프라인)

### 오프라인 라이선스 발급 모달

- 하드웨어 fingerprint 입력 (필수)
- 라벨 입력 (선택)
- 조직 선택 (선택)
- 만료일 설정:
  - 조직 미선택 시: Settings의 오프라인 기본 유효 기간이 기본값으로 표시. 수동 변경 가능
  - 조직 선택 시: 해당 조직 라이선스의 offlineMaxDays 기준으로 만료일이 자동 계산되어 표시. 수동 변경 가능
- 하단: "취소" / "발급"

### 오프라인 비활성화 코드 입력 모달

- 비활성화 코드 입력 필드
- 하단: "취소" / "폐기 처리"

## 주요 인터랙션

- **"+ 오프라인 라이선스 발급" 클릭**: 발급 모달 열림
- **발급 모달 "발급"**: 라이선스 키 생성 → 키를 다이얼로그에 표시 (복사 가능)
- **온라인 "폐기" 클릭**: 확인 다이얼로그 → 즉시 폐기 + 토스트
- **오프라인 "폐기" 클릭**: 비활성화 코드 입력 모달 열림 → 코드 입력 → 폐기 처리 + 토스트
- **"키 복사" 클릭**: 클립보드에 복사 + 토스트
- **"삭제" 클릭**: 폐기 상태인 것만 삭제 가능 → 확인 다이얼로그

## 상태별 화면

- empty: "등록된 디바이스가 없습니다" 안내
- 필터 결과 없음: "조건에 맞는 디바이스가 없습니다" 안내
