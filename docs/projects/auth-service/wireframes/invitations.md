# 초대 관리

## 진입 경로

- 사이드바 "Invitations" 클릭 → `/admin/invitations`

## 화면 구조

- 상단:
  - 조직 선택 드롭다운 (필수 — 초대는 특정 조직에 대한 것이므로 조직을 먼저 선택해야 한다)
  - "+ 새 초대" 버튼 (조직 선택 후 활성화)
- 메인:
  - 테이블: 이메일, 역할, 상태 (Pending/Accepted/Expired/Cancelled), 생성일, 만료일
  - Pending 행에 "취소" 버튼

### 초대 생성 모달

"+ 새 초대" 클릭 시 모달이 열린다.

- 이메일 입력 필드
- 역할 선택 드롭다운
- 유효 기간 설정 (시간 단위, 기본 72시간)
- 하단: "취소" / "초대 발송"

## 주요 인터랙션

- **조직 선택 변경**: 해당 조직의 초대 목록으로 갱신
- **"+ 새 초대" 클릭**: 생성 모달 열림 (조직이 선택된 상태)
- **"초대 발송" 클릭**: 초대 생성 + 이메일 발송 + 목록 갱신 + 토스트
- **"취소" 클릭**: 확인 다이얼로그 → Pending 초대 취소

## 상태별 화면

- 조직 미선택: "조직을 선택하세요" 안내
- 초대 empty: "발송된 초대가 없습니다" 안내
