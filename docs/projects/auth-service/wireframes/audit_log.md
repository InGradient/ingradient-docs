# 감사 로그

## 진입 경로

- 사이드바 "Audit Log" 클릭 → `/admin/audit`

## 화면 구조

### 상단 — 시각화 그래프

- 시간대별 이벤트 추이 라인 차트 — 이상 징후 발견용. 이벤트 유형별로 라인 색상 구분:
  - 인증: 로그인 성공, 로그인 실패, 로그아웃, 전체 로그아웃, 계정 잠금
  - 사용자: 사용자 생성, 사용자 수정, 사용자 삭제, 사용자 승인
  - 멤버: 멤버 추가, 멤버 제거, 멤버 역할 변경, 멤버 정지
  - 조직: 조직 생성, 조직 이름 변경, 조직 상태 변경
  - 디바이스: 디바이스 등록, 디바이스 폐기, 디바이스 삭제
  - 초대/가입: 초대 생성, 초대 수락, 가입 코드 사용
  - 비밀번호: 비밀번호 재설정 요청, 비밀번호 재설정 완료, 비밀번호 초기화
  - 라이선스: 라이선스 할당
  - 엔타이틀먼트: 엔타이틀먼트 발급, 엔타이틀먼트 취소, 엔타이틀먼트 갱신
  - SSO: SSO 로그인 성공, SSO JIT 계정 생성
  - 세션: 관리자에 의한 세션 종료
- 이벤트 유형별 분포 도넛/바 차트 (위 카테고리 기준으로 그룹핑)
- 날짜 범위 선택으로 그래프 기간 조절

### 하단 — 상세 로그 목록

- 필터:
  - 카테고리 드롭다운 (전체 / 인증 / 사용자 / 멤버 / 조직 / 디바이스 / 초대·가입 / 비밀번호 / 라이선스 / 엔타이틀먼트 / SSO / 세션)
  - 카테고리 선택 시 하위 이벤트 유형 드롭다운 활성화 (예: 인증 → 로그인 성공, 로그인 실패, 로그아웃, 전체 로그아웃, 계정 잠금)
  - 날짜 범위 (시작일, 종료일)
- 내보내기 버튼 (CSV, JSON)
- 테이블: 시각, 이벤트 유형, 사용자, 대상, 상세
- 페이지네이션

## 주요 인터랙션

- **그래프 구간 클릭**: 해당 시간대로 필터가 자동 적용됨 → 하단 로그 목록이 갱신
- **이벤트 유형 필터**: 드롭다운 선택 시 그래프와 목록이 함께 갱신
- **날짜 범위 변경**: 그래프와 목록이 함께 갱신
- **내보내기 클릭**: 현재 필터 조건 적용된 상태로 CSV 또는 JSON 다운로드
- **행 클릭**: 이벤트 상세 정보를 드로어 또는 확장 행으로 표시 (시각, 유형, 사용자, IP, 메타데이터)

## 상태별 화면

- empty: "이벤트가 없습니다" 안내 (그래프 영역 빈 상태)
- loading: 그래프 스켈레톤 + 테이블 스켈레톤
- 필터 결과 없음: "조건에 맞는 이벤트가 없습니다" 안내
