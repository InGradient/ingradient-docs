# 조직 관리

## 진입 경로

- 사이드바 "Organizations" 클릭 → `/admin/organizations`

## 화면 구조

### 조직 목록

- 상단:
  - 페이지 제목 "Organizations"
  - 요약 카드: 총 조직 수, 활성, 비활성
  - 검색바 (이름, 코드)
  - "+ 새 조직" 버튼
- 메인:
  - 테이블: 코드, 이름, 상태 (Active/Inactive), 멤버 수
  - 행 클릭 시 조직 상세 모달이 열림

### 조직 생성 모달

"+ 새 조직" 클릭 시 모달이 열린다.

- 조직 이름 입력 필드
- 조직 코드 (이름 입력 시 자동 생성 — 소문자 변환, 빈칸은 `-`로 치환. 예: "A 병원" → `a-병원`. 동일 코드가 있으면 `-2`, `-3` 자동 부여)
- 코드는 자동 생성 후 수동 수정도 가능
- 하단: "취소" / "생성" 버튼

### 조직 상세 모달

목록에서 행 클릭 시 모달이 열린다.

- 모달 상단:
  - 조직 정보 (코드, 이름 — 인라인 수정 가능, 상태 뱃지)
  - 닫기 (X) 버튼
  - 액션 버튼: "수정"/"저장", "상태 변경" (Activate / Deactivate)
- 탭:
  - **멤버**: 멤버 목록 테이블 (이름, 이메일, 역할, 가입일) + "+ 멤버 추가" 버튼
  - **라이선스**:
    - 플랜 코드
    - 최대 사용자 수 / 현재 사용자 수 (프로그레스바)
    - 최대 디바이스 수 / 현재 디바이스 수 (프로그레스바)
    - 오프라인 기능 허용 여부 (토글)
    - 오프라인 최대 유효 기간 (일 단위 — 오프라인 라이선스 발급 시 만료일 자동 계산에 사용)
    - 오프라인 유예 모드 (read_only / disabled)
    - 라이선스 만료일 (날짜 선택 — 만료되면 조직 전체 접근 제한)
    - 라이선스 상태 (Active / Suspended)
    - 하단: "Save" 버튼
  - **보안 설정**:
    - 상단: "시스템 기본값 사용" / "커스텀 설정" 라디오 선택
    - "시스템 기본값 사용" 선택 시: 현재 시스템 기본값을 읽기 전용으로 표시 (Settings 페이지에서 관리됨)
    - "커스텀 설정" 선택 시: 아래 항목이 수정 가능해짐 (시스템 기본값보다 강하게 설정 가능)
    - 인증 섹션:
      - MFA 필수 여부 (토글)
      - 셀프 가입 허용 여부 (토글)
      - 계정 잠금 — 실패 횟수 (숫자 입력)
      - 계정 잠금 — 잠금 시간 (분 단위)
    - 비밀번호 정책 섹션:
      - 최소 길이 (숫자 입력)
      - 대문자 필수 (토글)
      - 소문자 필수 (토글)
      - 숫자 필수 (토글)
      - 특수문자 필수 (토글)
      - 비밀번호 만료 기간 (일 단위, 0이면 만료 없음)
      - 비밀번호 이력 수 (이전 N개 비밀번호 재사용 금지)
    - 세션 제한 섹션:
      - 최대 동시 세션 수 (숫자 입력, 0이면 무제한)
      - 세션 타임아웃 (분 단위)
    - 하단: "Save" 버튼 (변경사항 있을 때만 활성화)

### 멤버 추가 모달

"+ 멤버 추가" 클릭 시 모달이 열린다.

- 사용자 검색/선택 (기존 사용자 중)
- 역할 선택 드롭다운
- 하단: "취소" / "추가" 버튼

## 주요 인터랙션

- **행 클릭**: 조직 상세 모달이 열림
- **"+ 새 조직" 클릭**: 생성 모달 열림
- **생성 모달 "생성" 클릭**: 코드 중복 검사 → 성공 시 모달 닫힘 + 목록 갱신 + 성공 토스트
- **멤버 탭 "역할 변경"**: 인라인 드롭다운으로 역할 변경 → 즉시 저장 + 성공 토스트
- **멤버 탭 "제거"**: 확인 다이얼로그 → 멤버 제거 + 목록 갱신
- **"상태 변경" 클릭**: 확인 다이얼로그 → Active/Inactive 토글

## 상태별 화면

- 목록 empty: "등록된 조직이 없습니다" 안내
- 멤버 탭 empty: "멤버가 없습니다. 멤버를 추가하세요" 안내 + 추가 버튼
- 라이선스 미할당: "라이선스가 할당되지 않았습니다" 안내
