# 사용자 관리

## 진입 경로

- 사이드바 "Users" 클릭 → `/admin/users`

## 화면 구조

### 사용자 목록

- 상단:
  - 페이지 제목 "Users"
  - 요약 카드: 총 사용자 수, 승인 대기, 활성, 비활성, 잠금
  - 검색바 (Login ID, 이름, 이메일로 검색)
  - 역할 필터 드롭다운 (All / Admin / Member)
  - 상태 필터 드롭다운 (All / Pending / Active / Inactive / Locked)
  - "+ 새 사용자" 버튼
- 메인:
  - 테이블: Login ID, 이름, 이메일, 역할 (뱃지), 상태 (뱃지 — Pending: 노랑, Active: 초록, Inactive: 회색, Locked: 빨강), 소속 조직, 마지막 로그인
  - Pending 행에는 "승인" 버튼이 인라인으로 표시됨
  - 행 클릭 시 사용자 상세 모달이 열림
- 하단:
  - 페이지네이션

### 사용자 생성 모달

"+ 새 사용자" 클릭 시 모달이 열린다.

- Login ID 입력 필드 (필수, 중복 검사)
- 비밀번호 입력 필드 (필수, 규칙 안내: 8자 이상, 문자+숫자)
- 비밀번호 확인 입력 필드
- 이름 입력 필드 (필수)
- 이메일 입력 필드 (필수)
- 조직 선택 드롭다운 (선택 — 비워두면 소속 없는 개인 사용자)
- Product 선택 (멀티셀렉트 — 다중 Product 배정 가능. 조직 선택 시 활성화)
- 선택된 Product별 역할 드롭다운 (각 Product 옆에 표시)
- 하단: "취소" / "생성" 버튼

### 사용자 상세 모달

목록에서 행 클릭 시 모달이 열린다.

- 모달 상단:
  - 사용자 이름 (큰 텍스트)
  - 상태 뱃지 (Active/Inactive/Locked)
  - 역할 뱃지 (Admin/Member)
  - 닫기 (X) 버튼
- 모달 하단: "Cancel" / "Save" 버튼 (변경사항 있을 때만 Save 활성화)

- 탭 1 — 기본 정보:
  - Login ID (읽기 전용)
  - 이름 (수정 가능)
  - 이메일 (수정 가능)
  - 역할 (수정 가능 — 드롭다운)
  - 상태 (읽기 전용)
  - 생성일 (읽기 전용)
  - 마지막 로그인 (읽기 전용)
  - 소속 조직 (수정 가능 — 드롭다운)
  - 소속 Product 목록 (Product 이름, 역할)
  - "+ Product 추가" 버튼 → Product 선택 + 역할 지정 인라인
  - Product별 역할 변경 (인라인 드롭다운)
  - Product에서 제거 (X 버튼)
  - 구분선
  - 승인 행: (Pending 상태일 때만 표시) 설명 텍스트 + "승인" 버튼 → 승인 시 Active로 전환
  - 비밀번호 초기화 행: 설명 텍스트 + "초기화" 버튼
  - 강제 로그아웃 행: 설명 텍스트 + "로그아웃" 버튼
  - 활성화/비활성화 행: 현재 상태 표시 + "비활성화" 또는 "활성화" 버튼
  - 잠금 해제 행: (Locked 상태일 때만 표시) 설명 텍스트 + "잠금 해제" 버튼
  - 구분선
  - 사용자 삭제 행: 경고 텍스트 + "삭제" 버튼 (빨간색)

- 탭 2 — 활동:
  - 활동 요약:
    - 마지막 로그인 시각 + IP
    - 총 로그인 횟수 (최근 30일)
    - 로그인 실패 횟수 (최근 30일)
    - 계정 잠금 이력 횟수
  - 최근 활동 목록 테이블:
    - 시각, 이벤트 유형 (뱃지: 로그인 성공/실패, 로그아웃, 비밀번호 변경, 역할 변경, Product 접근 등), IP, 상세
    - 이벤트 유형 필터 드롭다운
    - 날짜 범위 필터
    - 페이지네이션
  - 활성 세션 목록:
    - IP, 브라우저/OS (아이콘 + 텍스트), 생성 시각, 마지막 사용 시각
    - 현재 세션 표시 (뱃지: "현재")
    - 개별 세션 "종료" 버튼
    - "전체 세션 종료" 버튼

## 주요 인터랙션

- **행 클릭**: 사용자 상세 모달 열림
- **"+ 새 사용자" 클릭**: 생성 모달 열림
- **생성 모달 "생성" 클릭**: Login ID 중복 검사 → 비밀번호 규칙 검증 → 성공 시 모달 닫힘 + 목록 갱신 + 성공 토스트
- **상세 모달 기본 정보 "저장" 클릭**: 변경된 필드만 저장 + 성공 토스트
- **상세 모달 Product 추가/역할 변경/제거**: 인라인으로 즉시 반영
- **목록 "승인" 클릭 또는 상세 모달 "승인" 클릭**: 확인 다이얼로그 → Pending → Active 전환 + 토스트. 승인 후 사용자가 로그인 가능해진다
- **"비밀번호 초기화" 클릭**: 확인 다이얼로그 → 확인 시 임시 비밀번호 생성 → 임시 비밀번호를 다이얼로그에 표시 (복사 버튼 포함)
- **"강제 로그아웃" 클릭**: 확인 다이얼로그 → 확인 시 모든 세션 종료 + 성공 토스트
- **"비활성화" 클릭**: 확인 다이얼로그 ("이 사용자의 모든 세션이 종료되고 로그인할 수 없게 됩니다") → 상태 변경 + 토스트
- **"활성화" 클릭**: 확인 다이얼로그 → 상태 변경 + 토스트
- **"잠금 해제" 클릭**: 확인 다이얼로그 → 잠금 해제 + 토스트
- **"삭제" 클릭**: 위험 확인 다이얼로그 (빨간색, "이 작업은 되돌릴 수 없습니다") → 삭제 후 모달 닫힘 + 목록 갱신 + 토스트

## 상태별 화면

- 목록 empty: "등록된 사용자가 없습니다. 새 사용자를 생성하세요" 안내 + 생성 버튼
- 목록 loading: 요약 카드 스켈레톤 + 테이블 영역 스켈레톤
- 검색/필터 결과 없음: "조건에 맞는 사용자가 없습니다" 안내
- 생성 모달 유효성 에러: 해당 필드 하단에 에러 메시지 (빨간색)
- 생성 모달 Login ID 중복: "이미 사용 중인 Login ID입니다" 에러
- 상세 모달 활동 탭 empty: "활동 이력이 없습니다" 안내
- 상세 모달 세션 empty: "활성 세션이 없습니다" 안내
