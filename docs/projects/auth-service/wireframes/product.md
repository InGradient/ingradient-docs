# Product 관리

## 진입 경로

- 사이드바 "Products" 클릭 → `/admin/products`

## 화면 구조

### Product 목록

- 상단:
  - 페이지 제목 "Products"
  - 검색바 (이름, 코드)
  - 상태 필터 드롭다운 (All / Active / Inactive)
  - "+ 새 Product" 버튼
- 메인:
  - 테이블: 코드, 이름, 상태 (뱃지), 멤버 수
  - 행 클릭 시 Product 상세 모달이 열림

### Product 생성 모달

"+ 새 Product" 클릭 시 모달이 열린다.

- Product 이름 입력 (필수)
- Product 코드 (이름 입력 시 자동 생성 — 소문자 변환, 빈칸은 `-`로 치환. 예: "INGRADIENT Platform" → `ingradient-platform`. 동일 코드가 있으면 `-2`, `-3` 자동 부여)
- 코드는 자동 생성 후 수동 수정도 가능
- 상태 선택 (Active/Inactive)
- 하단: "취소" / "생성"

### Product 상세 모달

목록에서 행 클릭 시 모달이 열린다.

- 모달 상단:
  - Product 이름 (큰 텍스트)
  - 상태 뱃지 (Active/Inactive)
  - 닫기 (X) 버튼
- 모달 하단: "Cancel" / "Save" 버튼 (변경사항 있을 때만 Save 활성화)

- 탭 1 — 기본 정보:
  - Product 코드 (읽기 전용)
  - Product 이름 (수정 가능)
  - 상태 (수정 가능 — 드롭다운)
  - 생성일 (읽기 전용)
  - 구분선
  - Product 삭제 행: 경고 텍스트 + "삭제" 버튼 (빨간색)

- 탭 2 — 멤버:
  - 멤버 목록 테이블: 사용자 이름, Login ID, 이메일, 역할, 가입일
  - 멤버 검색 (이름, Login ID)
  - "+ 멤버 추가" 버튼 → 사용자 검색/선택 + 역할 지정 인라인
  - 멤버별 역할 변경 (인라인 드롭다운)
  - 멤버 제거 (X 버튼)

## 주요 인터랙션

- **행 클릭**: Product 상세 모달 열림
- **"+ 새 Product" 클릭**: 생성 모달 열림
- **생성 모달 "생성"**: 코드 중복 검사 → 성공 시 모달 닫힘 + 목록 갱신 + 토스트
- **상세 모달 "Save"**: 변경된 필드만 저장 + 토스트
- **"+ 멤버 추가" 클릭**: 사용자 검색 드롭다운 → 선택 → 역할 지정 → 추가 + 토스트
- **멤버 역할 변경**: 인라인 드롭다운 선택 시 즉시 저장 + 토스트
- **멤버 제거 (X)**: 확인 다이얼로그 → 제거 + 토스트
- **"삭제" 클릭**: 위험 확인 다이얼로그 (빨간색) → 삭제 후 모달 닫힘 + 목록 갱신 + 토스트

## 상태별 화면

- 목록 empty: "등록된 Product가 없습니다" 안내 + 생성 버튼
- 목록 loading: 테이블 스켈레톤
- 검색 결과 없음: "조건에 맞는 Product가 없습니다" 안내
- 멤버 탭 empty: "멤버가 없습니다. 멤버를 추가하세요" 안내 + 추가 버튼
