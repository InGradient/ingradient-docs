- 상태: Released
- Owner: Auth

## 버전 목표

INGRADIENT 전체 제품군의 공통 인증/인가 기반을 확립한다. 조직, 멤버십, 라이선스, 디바이스, 오프라인 엔타이틀먼트, 감사 로그까지 포함.

## 새 기능

### 인증 / 세션

- [x] 로그인 (Login ID + 비밀번호, Access Token + Refresh Token 발급)
- [x] 토큰 갱신 (Refresh Token으로 Access Token 재발급)
- [x] 로그아웃 (세션 취소, Refresh Token 폐기)
- [x] 세션 관리 (활성 세션 조회, 개별 세션 취소)
- [x] 토큰 검증 (downstream 서비스용 API)
- [x] 계정 잠금 (5회 실패 시 15분 자동 잠금)
- [x] 비밀번호 재설정 (forgot-password → 토큰 → 새 비밀번호)
- [x] Admin 로그인 (System Admin 전용, is_admin claim)
- [x] Bootstrap 인증 (서비스 간 통신용 BOOTSTRAP_SECRET)

### 사용자 관리

- [x] 사용자 CRUD (생성, 조회, 수정, soft delete)
- [x] 계정 상태 관리 (ACTIVE, DISABLED, INVITED, LOCKED)
- [x] System Admin 플래그 (isSystemAdmin, 역할 시스템과 독립)
- [x] Edge 자격증명 발급
- [x] 사용자 활동 조회

### 조직 / 프로젝트

- [x] 조직 CRUD (ACTIVE, SUSPENDED, DISABLED)
- [x] 프로젝트 CRUD (ACTIVE, DISABLED, ARCHIVED)
- [x] 프로젝트 멤버 관리 (추가/삭제, 프로젝트별 역할 부여)

### 멤버십

- [x] 조직 멤버십 (추가, 역할 부여, 정지, 삭제)
- [x] 멤버십 상태 관리 (ACTIVE, INVITED, SUSPENDED, EXPIRED)
- [x] 프로젝트 멤버십 (Membership 하위, 프로젝트별 역할)

### 역할 / 권한

- [x] 시스템 역할 (ADMIN, ORGANIZER, MEMBER — GLOBAL scope seed)
- [x] 역할 코드 매핑 (mapRoleCode: JWT role_code 변환)
- [x] 커스텀 역할 (ORGANIZATION / PROJECT scope)
- [x] 권한 관리 (resource:action 패턴, RolePermission 매핑)

### 초대 / 가입

- [x] Invitation (이메일 초대, 역할 지정, PENDING/ACCEPTED/EXPIRED/CANCELLED)
- [x] Join Code (공개 가입 코드, 최대 사용 횟수/만료 설정)

### 라이선스

- [x] 조직 라이선스 (maxUsers, maxDevices, offlineMaxDays, featuresJson)
- [x] 라이선스 사용량 조회
- [x] 오프라인 디바이스 라이선스 (fingerprint 기반 발급/취소)

### 디바이스

- [x] 디바이스 등록 (Device UID 기반, PENDING)
- [x] 디바이스 승인/취소 (ACTIVE / REVOKED)
- [x] 조직별 디바이스 목록, 삭제

### 오프라인 엔타이틀먼트

- [x] Entitlement 발급 (디바이스+사용자 바인딩, RS256 JWT)
- [x] Entitlement 취소 (JTI 기반)
- [x] 조직별 Entitlement 목록

### 감사 로그

- [x] Audit Log (login, member, device, invitation, license, entitlement 이벤트 자동 기록)
