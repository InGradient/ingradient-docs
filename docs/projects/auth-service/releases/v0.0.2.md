## 버전 목표

INGRADIENT 전체 제품군의 공통 인증/인가 기반을 확립한다. 조직, 멤버십, 라이선스, 디바이스, 오프라인 엔타이틀먼트, 감사 로그까지 포함.

### 완료된 항목 (이번 구현)

- ✅ 이메일 발송: SMTP provider + HTML 템플릿 + 재시도 (기본값 console)
- ✅ 라이선스 한도 검사: maxUsers (멤버십+초대+가입코드) + maxDevices (디바이스 등록)
- ✅ 오프라인 entitlement 갱신: `PATCH /entitlements/:jti/renew`
- ✅ 디바이스 하트비트: `POST /devices/:id/heartbeat`
- ✅ Rate limit Redis: 전역 100회/분 + ioredis + fail-open
- ✅ 감사 로그 조회: `GET /audit-logs` + `GET /audit-logs/export` + Admin UI
- ✅ API 페이지네이션: 전체 9개 목록 API `{ items, total, page, pageSize }` 형식
- ✅ 역할 시드: ORG_ADMIN, ORG_MEMBER 추가
- ✅ 본인 전체 로그아웃: `POST /auth/logout-all`
- ✅ Admin 세션 관리: `GET /sessions` + Admin UI 전체 세션 테이블
- ✅ 웹훅: CRUD API + HMAC 서명 발송 + exponential backoff 재시도 + 자동 비활성화
- ✅ SSO/OAuth: Authorization Code Flow (Google/Microsoft) + IdP 설정 CRUD + JIT provisioning
