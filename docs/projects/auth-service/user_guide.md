# auth-service User Guide

## 대상 사용자

- 조직 관리자
- 내부 운영자
- 통합 제품 개발자

## 시작하기

1. 조직 생성
2. owner 계정 생성
3. owner membership 부여
4. license 발급
5. 사용자 초대 또는 join flow 진행

## 주요 사용 흐름

### 조직과 사용자 셋업

- organization 생성
- owner user 생성
- owner role 부여

### invitation과 join flow

- invitation 발급
- 신규 사용자 로그인
- invitation 수락

### edge and offline

- device 등록
- offline entitlement 발급
- revoke와 재등록 확인

## 자주 묻는 문제

- 로그인은 되는데 권한이 기대와 다름
- invite가 만료됨
- account가 lock 됨
- offline mode가 license 정책으로 거부됨
