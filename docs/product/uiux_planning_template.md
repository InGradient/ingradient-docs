# UIUX Planning Template

## 목적
화면 설계와 사용자 경험 방향을 잡을 때 사용하는 UI / UX 기획 템플릿이다.

## 상단 메타데이터
- 화면명 또는 기능명
- 대상 사용자
- 상태
- Owner
- 관련 기능 spec 링크

## 본문 템플릿

### 1. 화면 목적
- 이 화면이 해결해야 하는 핵심 문제
- 사용자가 여기서 반드시 할 수 있어야 하는 행동

### 2. 주요 사용자와 맥락
- 누가 이 화면을 쓰는가
- 어떤 이전 단계에서 들어오는가
- 어떤 다음 단계로 나가는가

### 3. 핵심 행동
- primary action
- secondary action
- 위험한 destructive action

### 4. 정보 구조
- 상단 영역
- 좌측 패널 또는 필터
- 메인 콘텐츠
- 우측 detail / inspector
- 하단 상태 또는 action bar

### 5. 상태 설계
- empty
- loading
- success
- error
- partial failure
- permission denied

### 6. 반응형과 환경 차이
- desktop
- tablet
- compact width
- online / offline 또는 cloud / on-prem 차이

### 7. 접근성과 피드백
- 키보드 접근
- 포커스 이동
- status message
- destructive action confirmation

### 8. 연계 포인트
- 필요한 API
- 필요한 데이터
- 권한 영향
- release note 또는 user guide 영향
